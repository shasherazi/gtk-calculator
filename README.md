# GTK Calculator

GTK calculator is a simple arithmetic calculator app writthen in GTK4 using C.

## Installation

You will need:

- [GTK4](https://www.gtk.org/docs/installations/)
- A C compiler (GCC, Clang, etc.)

To build the app, run:

```bash
make
./calculator
```
